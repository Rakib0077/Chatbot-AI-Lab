# 🤖 AI Lab Chatbot

An advanced AI chatbot built in Google Colab using PyTorch and Hugging Face Transformers.

## Features

- Custom Intent Classification
- Open-Domain Chat (DialoGPT)
- Text Summarization (BART)
- PDF Reader & Summarizer
- Sentiment Analysis
- Voice Input & Output
- Gradio Web Interface

## Technologies

- Python
- Google Colab
- PyTorch
- Transformers
- Gradio
- NLTK
- gTTS
- VADER Sentiment
